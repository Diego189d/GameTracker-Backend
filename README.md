 GameTracker – Backend (MERN)

Este es el backend oficial del proyecto **GameTracker**, una plataforma creada para gestionar una biblioteca personal de videojuegos, permitiendo a los usuarios registrar juegos, escribir reseñas, llevar estadísticas y administrar su historial gamer.

Este backend está construido con la arquitectura **MERN (MongoDB, Express, React, Node.js)** y expone los endpoints necesarios para que el frontend pueda interactuar con la base de datos.

---

Características principales

✔ API REST completa  
✔ CRUD de Videojuegos  
✔ CRUD de Reseñas  
✔ Conexión a MongoDB Atlas  
✔ Código modular con controladores, modelos y rutas  
✔ Manejo de CORS  
✔ Variables de entorno con dotenv  
✔ Servidor Express limpio y escalable
